【游戏名称】 忍者突袭

【游戏介绍】 这是一款基于 Cocos 引擎开发的动作闯关小游戏（Cocos Creator 3.8.8）。游戏中，你需要控制忍者在关卡中移动与出击，规避危险并击败目标，在不断推进中获得更高分数并完成挑战。

【玩法说明】 游戏中玩家需要根据场景中的敌人分布与地形变化，调整移动与攻击节奏，击中目标并避免受到伤害，持续完成关卡推进。

道具： 本版本以基础动作闯关玩法为主，常见关卡元素包含敌人单位、障碍区域、边界、可移动路径与攻击判定（具体以游戏内实际显示为准）。

1. 角色起点： 每关开始时忍者会出现在指定位置，作为行动起点。
2. 目标敌人： 按关卡要求击败目标单位即可推进当前进度。
3. 危险区域： 场景中可能存在障碍或高风险区域，需要规划路线躲避。
4. 攻击节奏： 把握攻击时机与间隔，可提高命中效率并降低失误。
5. 走位与规避： 合理移动并规避敌方攻击，可以更稳定地完成关卡。

操作： 你可以通过鼠标点击/拖动（PC 端）或手指触控拖动（移动端）进行移动与出击操作。游戏为 web-mobile 构建版本，支持浏览器内交互体验。

目标： 在规则允许的条件下完成关卡目标，尽量减少失误与受击次数，提高通关效率与得分表现。

Tips： 先观察敌人位置与行动节奏再推进；优先处理威胁较高目标；遇到复杂场景时先走位再出击更稳。

【技术介绍】 使用基于 Cocos 引擎的 Cocos Creator 3.8.8 开发的 H5 游戏，主要语言为 JavaScript，构建平台为 web-mobile。项目入口为 index.html，运行时通过 SystemJS 加载 cocos-js/cc.js 与资源配置，设计分辨率为 1280 × 720，支持 PC 和移动端浏览器。
