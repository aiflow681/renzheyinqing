【游戏名称】
忍者突袭 - 引擎导出版

【游戏介绍】
这是一款使用游戏引擎导出的动作防御类小游戏。游戏中，你扮演一名忍者，面对从上方不断来袭的敌人武士。你需要通过投掷手里剑消灭敌人，保护自己的防线！本版本采用武士主题视觉风格，红黑金配色，日式装饰元素。

【玩法说明】
游戏中，敌人武士会从屏幕上方随机出现，不断向下移动攻击你。

障碍：
1. 敌人武士会从上方不断来袭，速度会随着得分增加而加快。
2. 如果敌人到达底部防线，你会失去生命值。
3. 生命值耗尽游戏结束。

操作：
点击或触摸屏幕 - 投掷手里剑攻击敌人。
手里剑会飞向点击的位置，击中敌人获得分数。

目标：
消灭尽可能多的敌人，获得更高的分数！

Tips：游戏结束会记录你的最高分数。
手里剑可以击中多个敌人，注意敌人速度会越来越快！

【技术介绍】
使用第三方自研游戏引擎开发（引擎版本1.0.3），主要语言是JavaScript，使用了Canvas 2D渲染技术，精灵动画系统，碰撞检测系统等。
